# fisicas

A physics engine port of [cyclone-physics](https://github.com/idmillington/cyclone-physics)
to Zig, developed with a Data-Oriented design. The engine is pure (no rendering
dependencies); demos use [raylib](https://github.com/raysan5/raylib) for both
native (OpenGL) and web (WebGL via emscripten) targets.

## Requirements

### Toolchain

- **Zig 0.16.0** (the engine and raylib 6.0 both target this version; 0.15.x
  will not build raylib's `build.zig`).

### Native demo (`zig build demo-native`)

Desktop build links raylib with the GLFW backend, which requires the X11
development libraries on Linux. Install them with:

```sh
sudo apt-get install -y libx11-dev libxrandr-dev libxinerama-dev libxcursor-dev libxi-dev libgl-dev
```

> Note: building in `Debug` may abort inside raylib's `GetCurrentMonitor`
> (`rcore_desktop_glfw.c`) due to signed-integer overflow on multi-monitor
> setups. Use `-Doptimize=ReleaseFast` (or `ReleaseSmall`) for the demo, where
> C integer overflow wraps as in standard C.

### Web demo (`zig build demo-web`)

Emscripten is **not** installed system-wide. The `emsdk` and `zemscripten`
packages are pulled in transitively by raylib 6.0 and fetched automatically
into `zig-pkg/` on the first `zig build --fetch`.

## Build

```sh
# Fetch dependencies (raylib, emsdk, zemscripten)
zig build --fetch

# Engine tests (f64)
zig build test

# Native demo (desktop, OpenGL) — run with ReleaseFast to avoid the overflow trap
zig build demo-native -Doptimize=ReleaseFast

# Web demo (WebGL via emscripten)
zig build demo-web
```

Build artifacts are placed under `zig-out/`. The `zig-pkg/` directory holds the
fetched dependency sources and is gitignored (regenerated by `zig build --fetch`).

## Problems encountered

### STOPSIG in zig 0.16.0

In zig 0.16.0 there is a bug when **STOPSIG** is called, originally it returns a `u32`, I fixed it in my local machine:

```zig
// zig-x86_64-linux-0.16.0/lib/std/os/emscripten.zig:214
    
    // old
    pub fn STOPSIG(s: u32) u32 {
        return @enumFromInt(EXITSTATUS(s));
    }

    // new
    pub fn STOPSIG(s: u32) SIG {
        return @enumFromInt(EXITSTATUS(s));
    }
```
